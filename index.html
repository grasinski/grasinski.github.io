  • Midnight Madness (Mac) (Windows)

    • For running on MacOS:
      1. Open terminal and navigate to the folder you unzipped the game to.
      2. cd MacBuild.app
      3. cd Contents/MacOS
      4. chmod u+x Midnight Madness (You can use tab to autocomplete so you don't have to try to type space in terminal.)
      5. ./Midnight Madness
      6. Click OK on the popup
      7. Open System Preferences -> Security & Privacy
      8. At the bottom you should be able to click "Open Anyway"
      9. The game should open!
